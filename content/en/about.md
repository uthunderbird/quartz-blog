I work as a **Lead Software Engineer at EPAM Systems**, where I specialize in code analysis through syntactic trees using LLM technology. My current focus is building tools for call graph construction, semantic analysis, business domain extraction, and feature implementation analysis. With 15 years of programming experience (since age 19, and programming enthusiast since age 11), I bring deep technical expertise in LLM integration, semantic code analysis, graph theory, API design and development.
