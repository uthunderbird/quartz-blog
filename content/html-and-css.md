# HTML and CSS Fundamentals

[[HTML and CSS]] form the foundation of web development. Understanding these technologies is crucial for anyone starting their [[Web Development Journey]].

## HTML - Structure

HTML (HyperText Markup Language) provides the structure for web pages:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
</head>
<body>
    <h1>Welcome to My Site</h1>
    <p>This is a paragraph of text.</p>
</body>
</html>
```

### Key HTML Concepts

- **Semantic Elements** - Using `<header>`, `<nav>`, `<main>`, `<footer>`
- **Forms** - Collecting user input
- **Accessibility** - Making content available to all users
- **SEO** - Optimizing for search engines

## CSS - Styling

CSS (Cascading Style Sheets) controls the visual presentation:

```css
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

h1 {
    color: #333;
    text-align: center;
}
```

### CSS Features

- **Selectors** - Targeting specific elements
- **Box Model** - Understanding margins, borders, padding
- **Flexbox** - Modern layout techniques
- **Grid** - Two-dimensional layouts
- **Responsive Design** - Adapting to different screen sizes

## Learning Path

1. **Start with [[HTML and CSS]] basics**
2. **Practice with [[CSS Grid and Flexbox]]**
3. **Learn [[Responsive Design]] principles**
4. **Move on to [[JavaScript]] for interactivity**
